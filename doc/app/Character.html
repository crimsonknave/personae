Character

The basic building block of personae, the character class represents a World of Darkness character. Specifically, it holds all the information that would be represented on a player’s character sheet.

All required attributes have default values.

The Storyteller user is considered to be an owner of every character.

Constants

VIRTUES

An Array of Strings of valid virtues in World of Darkness games. Primarily used to populate dropdown menus.

VICES

An Array of Strings of valid vices in World of Darkness games. Primarily used to populate dropdown menus.
